# oh-my-opencode 安装成功！ 🎉

恭喜！你已经成功安装了 OhMyOpenCode！

## 安装摘要

- **工具**: oh-my-opencode v3.12.3
- **配置**: 未配置任何付费模型提供商
- **备用模型**: opencode/big-pickle (免费模型)

## 重要提示

由于你没有配置任何高级模型提供商（如Claude、OpenAI、Gemini等），你将使用免费的备用模型。这可能会导致性能不如使用高级模型时的表现。

特别是 Sisyphus 代理强烈推荐使用 Claude Opus 4.6 模型，如果没有此模型，性能可能会显著下降：
- 编排质量降低
- 工具选择和委托能力较弱
- 任务完成可靠性较低

## 如何开始使用

在终端中输入以下命令即可开始使用：

```
opencode
```

要快速启动超工作模式，在提示中包含 `ultrawork` 或 `ulw` 关键词，所有功能都会像魔术一样工作——并行代理、后台任务、深度探索和不懈执行直到完成。

## 功能特色

- **并行处理**: 多个智能体同时工作
- **后台任务**: 持续运行的长时间任务
- **智能代理**: 自动选择最佳模型和工作流程
- **深度探索**: 自动探索代码库和文档
